This is a simple demo showing how to use different versions of the same library in the same project.
This projects imports three different versions of a library called "base library":
  • The oldest version 1.0.0 is part of this Angular project.
  • Version 2.0.0 is loaded indirectly by using another library.
  • There's also version 3.0.0, loaded by just another library.
